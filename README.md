## ğŸ“– How to Use

### Character Counter (Text Input)
1. Type in the "Enter Your Text" field
2. See real-time character count in the badge
3. View detailed statistics (words, letters, numbers, special chars)
4. Use **Copy** to copy text to clipboard
5. Use **Clear** to reset the input

### Message Counter (Textarea)
1. Type in the "Express Your Thoughts" field
2. Track your message up to 100 characters
3. Watch the progress bar fill as you type
4. Get color warnings when approaching the limit
5. Copy or clear your message with buttons

### Theme Toggle
- Click the moon/sun icon (top-right) to switch between light and dark modes
- Your preference is saved automatically
